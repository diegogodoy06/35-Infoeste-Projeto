Palestras / 35° edição

E-commerce, inovações em logística, sistemas de pagamento online, segurança digital, uso de inteligência artificial para recomendações de produtos, expansão em mercados internacionais

Mini Cursos
Introdução ao Desenvolvimento Web Front-end

Este curso introdutório aborda os fundamentos do desenvolvimento web front-end. Os participantes aprenderão HTML, CSS e JavaScript para criar páginas web interativas e responsivas. Tópicos Abordados: HTML básico, estilização com CSS, interatividade com JavaScript, design responsivo.

Escrevendo Código Limpo e Eficiente

Este curso é projetado para desenvolvedores de software de todos os níveis que desejam melhorar suas habilidades de programação, escrever código mais legível e eficiente e criar software de alta qualidade. Ao longo deste curso, você mergulhará nos princípios e métodos do Clean Code, aprendendo como transformar código confuso e difícil de entender em código limpo e organizado.

Desenvolvimento de Aplicativos Móveis com Flutter

Este curso se concentra no desenvolvimento de aplicativos móveis multiplataforma com o framework Flutter, do Google. Os participantes aprenderão a criar interfaces de usuário elegantes e funcionais para Android e iOS.

Desenvolvimento Ágil de Software: Práticas e Metodologias

Este curso abrangente sobre desenvolvimento ágil de software é projetado para profissionais de TI, gerentes de projeto e desenvolvedores que desejam dominar as práticas e metodologias ágeis para melhorar a entrega de software e maximizar a satisfação do cliente. Você aprenderá como implementar eficazmente abordagens ágeis em seus projetos e equipes.
